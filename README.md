# Next Generation Heat Pumps

Python toolbox to model advanced heat pump concepts (high temperature heat pumps, zeotropic, transcritical, ...)

Getting Started
---------------

Install CoolProp:

```bash
pip install CoolProp
```

Run the file "simple HTHP model.py"
